Methods

Following Chapter 3 of Morten Rand-Hendriksen's JavaScript Essentials on LinkedIn Learning, I have added methods to change the values of the properties:

  • name
  • color
  • volume
  • pocketNum

Access your browser's console to update the values of the properties using the added methods. Replace the asterisks (**) with the new value:

  • name: backpack.changeName(**);
  • color: backpack.changeColor(**);
  • volume: backpack.changeVolume(**);
  • pocketNum: backpack.changePocketNum(**);

EXAMPLE:

  1. backpack.changeName("Travel Backpack"); changes the value of the property name to "Travel Backpack". Calling the property with backpack.color results in the console displaying "Travel Backpack"
